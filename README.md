# AX 1on1 세션 Phase 1 — 형식지 추출 플러그인

1:1 AX 세션에서 신청자의 **원업무**를 *형식지*로 추출하는 스킬.

`[전]` 구글폼 받아 **세션별 인터뷰질문 생성** → (사람이 인터뷰) → `[후]` 인터뷰 raw 받아 **형식지 작성**.

> 인터뷰 진행 자체는 사람(AX 매니저)이 합니다. 스킬은 인터뷰를 *준비*([전])하고 *정리*([후])합니다.

## 설치

```
cd ~/.claude/commands
git clone {repo-url} 1on1-session-phase1-형식지
```

## 사용

새 Claude Code 세션에서 **`/형식지추출`** 입력.

- 매 실행 시 **자동 업데이트**(`git pull --quiet`). 네트워크 오류면 무시하고 진행.
- **세션 폴더 기준 단계 자동 감지**(+한 번 확인):
  - 인터뷰 raw 없음 → **인터뷰질문 생성 ([전])**
  - 인터뷰 raw 있음 → **형식지 작성 ([후])**

## 폴더 구조

```
1on1-session-phase1-형식지/
├── README.md
├── docs/
│   └── 사용자_가이드.md
└── skills/형식지추출/
    ├── SKILL.md
    └── references/
        ├── 형식지템플릿.md       (11칸 출력 구조)
        ├── 인터뷰프로토콜.md      (질문 은행 + 라이브 원칙)
        ├── 후처리규칙.md          ([후] 작성 규칙)
        ├── 구글폼슬롯매핑.md       (구글폼↔11칸 매핑·보강 규칙)
        └── 인터뷰질문양식.md       (인터뷰질문 출력 양식 = 콘솔)
```

세션 데이터는 `~/Desktop/AX-1on1-sessions/{기업·직무}_세션/`에 케이스별로 쌓입니다.

## 산출물

- `인터뷰질문_{신청자}_{업무}.md` — [전]. 인터뷰 때 보는 질문 (1개 파일)
- `Phase1_형식지_{신청자}_{업무}.md` — [후]. 11칸 형식지 + ★작성자 검토 필요 목록
