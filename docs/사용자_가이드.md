# 형식지추출 — 세션 담당자 사용 가이드

## 전체 흐름 (3단계)

1. **[전] 인터뷰질문 생성** — 구글폼 응답을 주면, *이 신청자에게 맞춘* 인터뷰질문을 만들어 줍니다.
2. **인터뷰** — 그 인터뷰질문 파일 *하나*만 보며 신청자와 1:1. 녹음/메모(raw)를 세션 폴더에 넣습니다.
3. **[후] 형식지 작성** — raw를 주면 11칸 형식지를 만들어 줍니다.

> 한 세션 = 한 폴더 (`~/Desktop/AX-1on1-sessions/{기업·직무}_세션/`). 스킬이 만들어 줍니다. 구글폼응답·인터뷰질문·raw·형식지가 여기 다 모입니다.

---

## 1. [전] 인터뷰질문 만들기

- `/형식지추출` 실행 → "인터뷰질문 만들까요?" → **예**
- **줄 것**: 구글폼 응답 (응답 스프레드시트 *다운로드 파일* 또는 해당 신청자 응답) + 첨부(양식 URL·파일) 있으면 같이
- **받을 것**: `인터뷰질문_{신청자}_{업무}.md` — **이 1개만 보고 인터뷰**하면 됩니다 (질문 흐름 + 어떻게 캐묻는지 + 진행 순서 다 들어 있음)
- 구글폼이 이미 답한 부분은 *가볍게 확인만*, 구글폼이 못 잡는 깊은 부분(**판단 기준·통과 기준·예외·시작/완료**)은 *꼭 묻도록* 짜여 있습니다.

## 2. 인터뷰

- `인터뷰질문_...md` 보며 진행. **들으러 가는 자리** — 적게 말하고 많이 듣기.
- 모호한 답("적당히"·"감으로")이 나오면 인터뷰질문에 적힌 *캐묻기*로 한 번 더 풀기.
- **녹음/메모(raw)를 세션 폴더에** 넣기 (다른 폴더에 있으면 [후]에서 경로/붙여넣기로 줘도 됩니다).
- 끝부분에 "제일 번거로운 지점은? / 바꿀 수 있다면?" 페인 질문이 있습니다 (Phase 2 신호).

## 3. [후] 형식지 만들기

- 다시 `/형식지추출` 실행 → "형식지 작성할까요?" → **예**
- **줄 것**: 인터뷰 raw (① 폴더 파일 / ② 경로 / ③ 붙여넣기 중 아무거나)
- **받을 것**: `Phase1_형식지_{신청자}_{업무}.md` — 11칸 형식지 + 끝에 **★ 작성자 검토 필요 목록**
- 대화창에도 **검토 필요 목록 + 미확인 항목**이 떠서 바로 확인 가능 (파일 안에도 동일하게 있습니다)

---

## 줄 것 / 받을 것 요약

| 단계 | 줄 것 | 받을 것 |
|---|---|---|
| [전] | 구글폼 응답 (+첨부) | 인터뷰질문 1개 파일 |
| 인터뷰 | (사람이 진행) | raw → 폴더에 |
| [후] | 인터뷰 raw | 형식지 + ★검토 목록 |

## 주의점

- **구현 방향(자동화·도구)은 형식지 본문에 안 들어갑니다** — 11번 "Phase 2 참고 메모"로 분리됩니다 (그건 Phase 2 몫). 형식지는 *지금 어떻게 일하는지*만 담습니다.
- **첨부(실제 양식·결과물)를 주면** 더 정확한 질문·형식지가 나옵니다 (실물 기반).
- **못 채운 항목은 지어내지 않고** 10번 "미확인"에 정직하게 남깁니다 → 후속 확인하면 됩니다.

## Phase 2 인계

- 형식지 **11번(보충 메모)**이 Phase 2(세션스킬)의 입력이 됩니다. 구현 의도·페인·권한 제약이 여기 모입니다.
